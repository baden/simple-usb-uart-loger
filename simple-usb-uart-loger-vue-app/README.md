## Відлагодження


```bash
nvm install --lts
nvm use --lts
npm install
npm run dev
```

## Збірка

```bash
nvm install --lts
nvm use --lts
npm install
npm run build
```

## Публікація

```bash
nvm install --lts
nvm use --lts
npm install
npm run build && npm run deploy
```
